About

The main objective of the Skeletonizr project is to create prototypes that serve as examples to get into the new technologies and web standards. For this purpose, this tool will be able to choose which technologies and which model to use.

Among the latest technologies we can find Node.js, Django on the backend, MongoDB and CouchDB in the database and HTML5-Boilerplate, Backbone.js, among many others.

All projects generated by Skeletonizr are ready to be installed and executed.

The Skeletonizr source files are written in Python and hosted on Github.

Goals

Skeletonizr is far from finished, but it also has a great start. Some goals of the project are:

  • Generate the server/backend source code
  • Generate the client/frontend source code
  • Customize the CRUD entities to generate the stubs
